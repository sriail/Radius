Popup & New Tab Interception Test

Note: When this page is loaded in Radius proxy, all attempts to open new tabs or windows should be intercepted and loaded in the main iframe instead.

Test 1: window.open() with URL

Expected: Should load example.com in the main iframe, not a new tab.

Test 2: Link with target="_blank"

Open Example.com in New Tab (link with target="_blank")

Expected: Should load example.com in the main iframe, not a new tab.

Test 3: window.open() without URL

Expected: Should be intercepted and ignored (about:blank).

Test 4: Multiple window.open() calls

Expected: Should load the first URL in the main iframe (second call may be blocked).

Test 5: Link with target="_new"

Open Example.com with target="_new"

Expected: Should load example.com in the main iframe, not a new window.

Test 6: Form with target="_blank"

Expected: Should load the form action URL in the main iframe, not a new tab.
